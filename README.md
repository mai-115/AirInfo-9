# AirInfo

A Flask dashboard for exploring live flight data and AI-powered insights for **Indian airlines**.

Enter an Indian airline IATA code (e.g. `AI`, `6E`, `SG`) along with your AviationStack API key to view schedules, statuses, route charts, and Gemini-generated insights.

Supported airlines include Air India, IndiGo, Vistara, SpiceJet, Air India Express, AirAsia India, Alliance Air, Akasa Air, and more.
